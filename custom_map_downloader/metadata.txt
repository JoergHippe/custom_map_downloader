about=Custom Map Downloader exports visible QGIS layers extent-based: pick a layer, choose an output CRS, set ground sampling distance (map units per pixel) or target scale and export georeferenced rasters. Pixel width/height are derived from extent and GSD; world files are always written. Optional VRT mode writes tiled GeoTIFFs plus a portable VRT instead of a merged raster. MBTiles mode writes an exact-extent Web Mercator tile pyramid with configurable zoom range, tile size, padding, tile estimates and SQLite metadata. Designed for XYZ/WMTS/WMS, raster and vector layers with progress feedback and automatic layer loading for raster outputs.
